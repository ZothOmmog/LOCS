# url/user/login/postLogin
- ***валидация данных из формы входа***
 Формат POST ***React->Сервер***
 Данные входа:
 

		{ 
			"Login": 
			{
				"mail": "String",
				"pas": "String"
			 }
		}
			
 
- ***Ответ на корректность данных***
 Формат JSON ***Сервер->React***
 Данные входа:
 

		{ 
			"Login":
			{
				"Flag": "Bool"
			}
		}

# url/user/SearchUser 
- ***Поиск пользователей***
 Формат POST ***React->Сервер***
 Данные поиска:

		{
			"nick"	: "String"
		}

- ***Ответ***
 Формат JSON ***Сервер->React***
 Данные поиска:
 
		[
			{
				"user": {
					"id_user": "Int",
					"nickname": "String"
				}
			},
		]
