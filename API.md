###### **url/user/login 
	**страница входа
---
# url/user/login/postLogin
- ***валидация данных из формы входа***
 Формат POST ***React->Сервер***
 Данные входа:
 

	{ 
			"Login": 
			{
				"Mail": "String",
				"Password": "String"
			 }
 	}
			
 
- ***Ответ на корректность данных***
 Формат JSON ***Сервер->React***
 Данные входа:
 

		{ 
			"Login":
			{
				"Flag": "Bool"
			}
		}

# url/user/registration/postreg 
- ***Регистрация***
 Формат POST ***React->Сервер***
 Данные регистрации:

		{ 
			"Registration":
			{
				"mail": "String",
				"nick": "String",
				"pas": "String"
			}
		}

- ***Ответ на корректность данных***
 Формат JSON ***Сервер->React***
 Данные входа:
 

		{
			"Login":
			{
				"NickNameFlag":"Bool",
				"MailFlag": "Bool"
			}
		}
