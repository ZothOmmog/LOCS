# url/upload/PhotoAcc
- ***загрузка фото в личный кабинет (пользователь авторизирован)***
 Формат POST ***React->Сервер***
 Данные входа:
 
			(form-data)
			
			{ 
				file : "(png, jpg, jpeg)"
			}
			
 
- ***Ответ на корректность данных***
 Формат JSON ***Сервер->React***
 Данные входа:
 

			{
				load: true
			}

при ошибке формата:

			{
				errFormat: "фото (png, jpg, jpeg) должно быть не больше 6 мб"
			}
